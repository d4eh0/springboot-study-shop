## 📌 What I Learned
- Spring Boot 기반 웹 서버와 REST API 설계

- 예외 처리 및 DTO 적용

- **Language**: Java 17

- **ORM**: JPA / Hibernate

- **Security**: Spring Security, JWT, Session
- **API**: REST API
- **Cloud**: AWS (EC2, S3)
